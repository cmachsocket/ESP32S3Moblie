# ESP32S3Moblie

自制的一个基于waveshare esp32s3-geek的一个移动设备，主要方便自己上学用

高中课业繁忙，暂时停止维护

使用 [tftziku](https://github.com/StarCompute/tftziku) 支持显示中文，但是经过了一些定制，暴露了一些接口

## 支持功能：

1. 背光调节

2. 亮/关屏记录时间到Flash,一定程度防止无网络情况下暴毙

3. 检测环境温度/气压/湿度/气体阻值

4. 背景

5. 熄屏（深度睡眠）和自定义熄屏时间 

6. 从网络获取天气

7. 联网

8. 从网络获取“一言”（句子）

9. 查看SD卡内图片

10. 阅读电子书
